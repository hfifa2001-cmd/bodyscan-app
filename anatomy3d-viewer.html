3D 근육 뷰어 — Z-Anatomy 연동

변환한 Z-Anatomy .glb를 불러오면 회전 가능한 3D 근육 모델이 뜨고, 근육 메쉬 이름으로 과부하=빨강·약화=파랑을 칠합니다. 이전 자세 분석의 과부하/약화 목록을 그대로 이 모델에 적용할 수 있어요.

Z-Anatomy → .glb 변환 방법 (이 단계만 직접 하시면 됩니다)
  1. Z-Anatomy 파일 다운로드 — z-anatomy.com 또는 GitHub의 Blender .blend 파일. (라이선스 CC BY 4.0 — 출처 표기 필요)
  2. Blender에서 열고, 컬렉션에서 Muscular system(근육)만 남기고 뼈·혈관·신경 등 다른 시스템은 숨기거나 삭제해 가볍게 만들기.
  3. (선택) 무거우면 각 오브젝트에 Decimate 모디파이어로 폴리곤을 줄이기. 웹에선 합쳐서 수십만~백만 폴리곤 이하 권장.
  4. 오브젝트 이름이 라틴 해부명(예: Trapezius.l, Gluteus maximus.r)인지 확인 — 이 이름으로 색칠을 매칭해요.
  5. File → Export → glTF 2.0 (.glb) 선택. Format=glb, Include=Selected/Visible Objects. 압축(Draco)은 끄는 게 호환에 안전해요.
  6. 나온 .glb를 위의 모델 불러오기로 열면 끝. (이 페이지를 다운로드해 브라우저로 직접 열면 인터넷 없이도 동작해요.)
이 뷰어는 모델을 당신 기기에서만 읽어요(업로드 없음). 자세 분석 결과 연결: 콘솔/코드에서 applyResult(['trapR','qlL'], ['rhombL']) 형태로 호출하면 해당 근육이 색칠됩니다. 의료 진단이 아닌 참고용이에요.
